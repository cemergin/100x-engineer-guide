# Episode L1-M05: PostgreSQL From Zero

## In This Episode
You'll run PostgreSQL in Docker, design the TicketPulse relational schema from business requirements, populate it with realistic data, and write your first JOIN queries. By the end, you'll have a working database with 7 tables and the confidence to read any schema diagram.

## Key Concepts
- Running PostgreSQL in Docker and connecting via `psql`
- Relational schema design: tables, types, constraints, foreign keys
- Data modeling from business requirements
- SQL fundamentals: SELECT, INSERT, JOIN, GROUP BY
- Reading EXPLAIN output and schema diagrams

## Prerequisites
- Docker installed and running (`docker --version` returns a version number)
- Basic terminal comfort (cd, ls, copy-paste commands)
- If you can run `docker compose up -d` without hesitation, you're ready
